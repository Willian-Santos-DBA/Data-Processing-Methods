# 🛠️ Métodos de Tratamento de Dados (MTD)

Este repositório é focado em técnicas de **Data Quality** e **Data Wrangling**. O objetivo principal é garantir a integridade dos dados através de métodos avançados de limpeza e transformação, permitindo que modelos de ML e relatórios de BI operem sobre bases confiáveis.

## 🚀 Tecnologias e Técnicas
- **Linguagem:** Python 3.x
- **Bibliotecas:** Pandas, Scikit-Learn (SimpleImputer), Matplotlib/Seaborn (Visualização).
- **Processos:** Imputação de dados (estratégias de média e moda), tratamento de valores ausentes, codificação de variáveis e normalização de pipelines.

---

## 📈 Projetos e Atividades Finalizadas

### 1. Análise de Mercado: Electric Vehicle Population (EV Market)
- **Contexto:** Estudo exploratório utilizando o dataset real da população de veículos elétricos (Washington State/Kaggle).
- **Foco:** Limpeza inicial, análise de distribuição de fabricantes e estruturação de dados para identificar tendências de adoção de tecnologia sustentável.
- **Arquivo:** `MTD_30_04_26_Final.ipynb`

### 2. Engenharia de Atributos e Imputação Avançada
- **Tratamento de Nulos:** Implementação técnica utilizando `SimpleImputer` para preenchimento inteligente de dados faltantes, diferenciando estratégias para variáveis numéricas e categóricas.
- **Data Integrity:** Garantia de que o conjunto de dados final esteja livre de inconsistências que poderiam gerar viés em modelos de predição de preços ou crédito.
- **Arquivo:** `MTD_07_05_26_Final.ipynb`
